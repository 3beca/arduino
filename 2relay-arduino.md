# Arduino to Relay 2CH
## Schema

```code
Arduino PIN 2 -> Relay ESP Rx
Arduino PIN 3 -> Relay ESP Tx
Arduino GND -> Relay ESP Gnd
Arduino 5v -> Relay Vin+
Arduino GND -> Relay Vin-
```
